# Next.js Tailwind Template

A clean starter for Next.js, TypeScript, and Tailwind CSS.

## Scripts

```bash
npm run dev
npm run build
npm run lint
```

## Included

- Next.js app router
- TypeScript
- Tailwind CSS
- Small `cn()` utility for conditional class names

Project-specific APIs, SEO metadata, auth, UI libraries, and icon libraries are
intentionally left out so each new project can add only what it needs.
